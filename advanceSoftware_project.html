Rent It Out

Advance Software Project

Description

"Rent It Out" is a car rental platform designed to connect users with vehicles for rent seamlessly. The project follows a modular architecture with a clear folder structure (routers, models, controllers) for better maintainability and scalability. It leverages RESTful APIs to provide secure and efficient communication between the frontend and backend.

Technologies Used

  • Backend: Node.js with MySQL (XAMPP)
  • Frontend: Flutter
  • API: RESTful API architecture
  • API Testing: Postman
  • Database Management: MySQL

Features

  • User-friendly car rental interface
  • Secure and modular backend operations
  • Responsive mobile-first design using Flutter
  • Comprehensive CRUD operations for cars and rentals
  • Seamless API integration with thorough testing
